Features for Admin

The following features are added for Odoo Marketplace Admin and seller. Some features are only accessible to admin to provide better control on Auctions in Odoo.

Run Auction On Your Odoo Marketplace

  • Hold Odoo e-Auctions on your marketplace website.
  • Create product auctions for any product from the Odoo backend.
  • The sellers can also create auctions (if allowed from the settings)

Option to Sell product Directly or Auction Away

  • Set an initial start price of the product for the auction.
  • Set a Reserve price (minimum selling price for the auction product.) to ensure profit.
  • Or You can also set a direct purchase price for an auction product for regular customers.

Create and Set Increment rules for Auto-Bid in an Odoo auction

  • Facilitates the admin to create bid increment rules for Auto-bid features.
  • The Sellers can only view the increment rules but cannot edit or create them.
  • The increment rules are then added to the product auction while creating it in Odoo

Enable/disable Auction Rights For Individual Sellers

  • Use Global Setting to Allow/disallow Auction creation rights for all sellers.
  • You can also enable/disable the rights for particular sellers from seller forms.
  • This feature is only for Marketplace admin. Sellers cannot access it

Deal In Different Types of products on Marketplace

  • Increase your product catalogue from regular products to auctionable products.
  • Use general products to sell directly on your Odoo Marketplace.
  • Also, Bring in Antiques and special products and auction them away to bidders from the Odoo Marketplace.

Auction State visible in Odoo backend

  • Check current stage/state of an Odoo Auction in the backend auction form.
  • The stage of auction changes in the backend as auction proceeds.
  • Such as Confirmed, running, extended, expired, etc.

Enable different notifications types to Odoo Auction subscribers

  • You can send different types of automatic email notifications to bidders.
  • Notify The Auction winner and/or Loser via email.
  • Send bidding related mail notifications like bid start, auction extended, new bid placed, etc.
  • Enable Auto-Bid mail Notifier to remind bidders about new auto-bids.

See Complete Bidders list in Product Auction form in backend

  • The bidders list is stored in the backend for each auction.
  • View the bidders list and their bids in each marketplace Auction form in the backend.

Presenting Different Types Of Bids - Simple & Auto bid

Simple Bid

  • Simple Bids Are the manual bids placed by the customer in an Auction.
  • There is no limit set to the manual bid. The customer can go as high as he wants.

Auto-Bid

  • If a customer opts for Auto-bid, then he can only increment his bid as per the increment rules set by the admin.
  • For Auto-bid, the system controls the bid increments of the customer as per the increment rules to make him the highest bidder by defined amount.
  • Say, the current highest bid is $500, and the increment rule allows for a $20 increment in auto-bid for $400-$ 650. So the customer will be able to place $520 as his next bid which then becomes the highest bid.
  • If now a higher bid say $580 is placed by some other customer, the system will allow the bid to be increased for auto-bid to $600 again making this the highest bid.
