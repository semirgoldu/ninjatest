Can this theme be used with Odoo Enterprise?

Yes, this theme is tested with Odoo Community as well as Enterprise Edition. for all the mentioned versions. We guarantee its compatibility for Odoo's default addons for both CE and EE except any third party modules/apps/themes

Multi Website Feature is available with this theme?

Yes, Odoo's Multi website feature is compatible with this Theme. You can configure this theme in multiple websites in the same database.

How can I do customization in this theme?

Our theme is customizable by using in-built building blocks and snippets. There are a number of headers, footers, pages to make a nice piece of website for your business. And in case you want any assistance in that, feel free to contact our technical team in email: support@bizople.com

Can I use demo data with this theme?

This demo data you see in the Live Demo is just for showing purposes. It's not given when you purchase the theme.

Are these images in your theme reusable?

No, the images loaded in the theme are copyrighted. In case you want to generate the images related to your business and website, feel free to contact us at our email: support@bizople.com

Is this theme resellable?

No. You can not resell this theme nor can you distribute it without the written consent of Bizople Solutions Pvt. Ltd.

Do I get a free upgrade for this theme if I have purchased it in the past?

Yes, we do. You get a free upgrade for the theme for the same Odoo version. You can compare the changelog file with your installed theme version and download the latest version in case you're using the older version.

Moreover, if you want any feature which is not needed in even the latest theme version, feel free to contact us at our email: support@bizople.com

If you are looking to download the theme for different Odoo versions then please note that starting version 13, every version of the theme will be sold separately.

Do you have any type of the support with the Theme?

We will provide you 3 months of support of bug fixes and security fixes with the purchase of the theme.

Do you have guidelines for installing this theme?

Yes, we have a user guide with the installation and configuration steps, please contact our support team at support@bizople.com

Does the theme support Multi-languages?

Yes, it is a Multi-language theme and it support different language(s) as per your need.

Do you personalize theme as per industry?

Yes, you can share your requirements to support@bizople.com and our team will get back to you as soon as possible.
